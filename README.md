# Specs :art:

- [x] [wlogout](https://github.com/ArtsyMacaw/wlogout) black/white theme ([HyDE](https://github.com/HyDE-Project/HyDE) inspired)
- [x] [waybar](https://github.com/Alexays/Waybar) black/white theme (personal)
- [x] [rofi](https://github.com/davatorium/rofi), currently by [adi1090x](https://github.com/adi1090x/rofi)
- [x] [dunst](https://github.com/dunst-project/dunst) notification manager (personal)
- [x] [lazyvim](https://github.com/LazyVim/LazyVim) (almost origin)[^1]
